# Professional Portfolio - Electrical BIM Engineer & Developer

This is a modern, high-performance portfolio template designed to showcase your work in **Electrical Design** and **Revit Add-in Development**. It includes a dynamic "Add Project" feature and a sleek dark mode design suitable for engineering tech.

## üöÄ How to Use

1.  **Open the Portfolio**: Simply open `index.html` in any modern web browser (Chrome, Edge, Firefox).
2.  **Customize Your Data**:
    *   Open `js/data.js` in a text editor.
    *   Update the `portfolioData` object with your real information (Name, Title, Links, Skills).
    *   **CV Integration**: Since the automated tool could not read your PDF directly, please manually copy the text from your CV into the `about` section or create a summary in `js/data.js`.
3.  **Add Projects**:
    *   You can add projects temporarily using the floating "+" button on the site.
    *   To save them permanently, copy the "JSON Code" shown in the alert after adding a project, and paste it into the `projects` array in `js/data.js`.
